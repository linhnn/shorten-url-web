# Shorten Url Web

Create React apps with Redux, Apollo.
Author: linhvt22@gmail.com

## Requirements

* node `^10.0.0`
* npm `^6.0.0`

## Running

```sh
npm install
npm start
```

Then open [http://localhost:3000/](http://localhost:3000/) to see app.

## Docker

* Run:

    ```
    docker-compose up -d
    ```
